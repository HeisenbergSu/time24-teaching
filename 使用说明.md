# 24时计时法教学系统 - 使用说明

## 快速开始

### 第一步：初始化数据库

双击运行 `init_db.bat` 或在命令行中执行：

```bash
cd backend
python init_data.py
```

这将创建数据库并初始化所有题目数据。

### 第二步：启动后端服务

双击运行 `start_backend.bat` 或在命令行中执行：

```bash
cd backend
python app.py
```

后端服务将在 `http://localhost:5000` 启动。

### 第三步：启动前端服务

打开新的命令行窗口，双击运行 `start_frontend.bat` 或在命令行中执行：

```bash
cd frontend
npm install  # 首次运行需要安装依赖
npm run serve
```

前端服务将在 `http://localhost:8080` 启动。

### 第四步：访问系统

在浏览器中打开 `http://localhost:8080` 即可使用系统。

## 功能说明

### 1. 课前探索模块

- **旧知复习**：包含4道一、二年级的时间知识题目
- **情景导入**：可播放/暂停的时钟动画，展示24小时的流转

### 2. 新知学习模块

- **互动时钟**：
  - 地球、太阳、月亮的场景展示
  - 可手动调节时间（0-23点）
  - 观察时针转动和背景变化
- **时间对比**：早上8点 vs 晚上8点的直观对比

### 3. 生活应用模块

- **时间尺**：
  - 24小时的时间尺展示
  - 点击任意时间查看12时和24时计时法的对应关系
- **转换规则**：清晰展示转换规则和示例
- **生活场景练习**：6道结合实际场景的练习题

### 4. 成就挑战模块

- **综合测验**：5道综合题目
- **成绩展示**：完成所有题目后显示成绩和评价
- **排行榜**：查看全班排名

### 5. 教师后台

- **数据统计**：
  - 总学生数、总答题数、正确答题数
  - 平均得分
  - 各模块统计（正确率）
- **学生成绩**：所有学生的详细成绩列表
- **错题分析**：
  - 按错误率排序
  - 显示错误学生数
  - 帮助教师精准定位教学难点

## 常见问题

### Q: 前端页面显示空白？

A: 确保：
1. 后端服务已启动（http://localhost:5000）
2. 前端已安装依赖（npm install）
3. 浏览器控制台没有报错

### Q: 无法加载题目？

A: 确保：
1. 已运行 `init_data.py` 初始化题目数据
2. 数据库文件 `time24_teaching.db` 已创建
3. 后端服务正常运行

### Q: 教师登录失败？

A: 默认账号：
- 用户名：`teacher`
- 密码：`teacher123`

如果仍然失败，检查数据库中的教师表是否已创建。

### Q: 如何添加新题目？

A: 
1. 编辑 `backend/init_data.py`
2. 在对应的题目列表中添加新题目
3. 重新运行 `python init_data.py`

## 技术说明

### 端口配置

- 后端：5000
- 前端：8080

如需修改，请编辑：
- 后端：`backend/app.py` 中的 `app.run(port=5000)`
- 前端：`frontend/vue.config.js` 中的 `devServer.port`

### 数据库

使用SQLite数据库，数据库文件位于 `backend/time24_teaching.db`

### 数据持久化

所有学生的答题记录和成绩都会保存在数据库中，教师可以随时查看。
